# 📊 Sales Performance Dashboard

## 🧠 Project Overview
This project analyzes sales data to generate business insights using SQL, Excel, and Power BI.

The dashboard helps stakeholders understand sales trends, top-performing products, and regional performance.

---

## 🛠️ Tools & Technologies
- SQL (MySQL)
- Excel
- Power BI

---

## 📁 Dataset
- 1000+ sales records
- Includes Order Date, Region, Category, Sales, Profit

---

## 🧹 Data Cleaning (Excel)
- Removed duplicates
- Handled missing values
- Created new columns:
  - Year
  - Month
  - Profit Margin

---

## 🗄️ SQL Analysis

### Key Queries:
- Total Sales & Profit
- Sales by Region
- Top 10 Products
- Monthly Sales Trend

---

## 📊 Dashboard Features (Power BI)
- KPI Cards (Sales, Profit, Margin)
- Sales Trend (Line Chart)
- Region-wise Performance
- Top Products
- Interactive Filters (Slicers)
